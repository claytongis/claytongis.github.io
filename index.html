Key Projects undertaken

I developed an open-source Data Hub Portal to enhance access to high-quality spatial data to support decision-making. The hub integrates three automated portals, the portals employ advanced machine learning and spatial analysis techniques to efficiently generate, analyse, and produce real-time geo-data.

After working closely with local authorities, I realized they lacked cost-efficient data management systems. As part of my academic research, I decided to address this challenge by designing a fully automated information management system. This system includes: A WebGIS portal, mobile application for data collection and Integration with desktop GIS (QGIS). The data is synced in real-time, ensuring efficient and up-to-date information management.

Zambia Red Cross faced a challenge in tracking volunteers throughout the country. To address this, I created a dashboard to automatically track volunteers from the branch level. The branches report through a mobile application, with the data being synced and analyzed in real time.
